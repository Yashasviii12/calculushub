# CalculusHub Extrema Finder - Algorithm

• Defined the problem of finding absolute minima and maxima on closed intervals using Python

• Converted user input strings to mathematical expressions using SymPy's sympify function

• Applied symbolic differentiation using SymPy's diff function to compute derivatives

• Solved f'(x) = 0 equation using SymPy's solve function to find critical points

• Evaluated function at critical points and endpoints using NumPy's lambdify for fast numerical computation

• Compared all values using Python's built-in min and max functions to identify global extrema

• Created interactive user interface with Python's input and output functions

• Integrated SymPy for symbolic math, NumPy for numerical evaluation, and Python's standard functions to deliver results
